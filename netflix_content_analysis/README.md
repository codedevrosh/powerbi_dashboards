# Netflix Content Analytics Dashboard 

## Project Overview

Designed and developed a multi-page interactive Netflix Content Analytics Dashboard using Power BI.  
The dashboard analyzes Movies and TV Shows to uncover trends in release patterns, genres, ratings, duration, seasons, and country of origin.

The project demonstrates data cleaning, transformation, DAX calculations, and structured dashboard design.

## Key Insights

- Significant growth in content production after 2015.
- Movies dominate overall catalog volume.
- Drama and International genres are most frequent.
- Majority of TV Shows have 1–2 seasons.
- TV-MA is the most common content rating.
- United States contributes the highest number of titles.
